markdown 常用命令

Jan 28, 2018 | tools

markdown优点:

  • 1.兼容了打开方式(所有人几乎都可以打开)
    word可能会打不开,所以选用.txt格式
  • 2.兼容了样式显示
    .txt没有样式
  • 3.转html方便

简单使用教程:

https://www.jianshu.com/p/q81RER

常用命令

1.标题

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

2.列表

格式:"- 文本"
例入:- 1.杨一真帅

3.链接

格式:[链接显示的文本](链接的地址)
例如:[百度](www.baidu.com)

4.图片

格式:![](图片链接)
例如:![](http://f.hiphotos.baidu.com/image/pic/item/503d269759ee3d6db032f61b48166d224e4ade6e.jpg)

5.粗体&斜体

斜体格式:*文本内容*
例如:*杨一真帅!*
粗体格式:**文本内容**
例如:**杨一真帅**

6.代码引用

多行格式:'''代码'''
单行格式:'代码'
例如:'hello word!'

7.表格

8.引用

格式:> 引用内容
例如:> 英雄必横空出世,且不问出处

9.代码块

<pre></pre>
